⚙️ Setup Dompet Keluarga

Ikuti 5 langkah berikut untuk menyambungkan Dompet Keluarga ke Google Sheets kamu

1
Buat Google Sheet baru
Klik tombol di bawah. Login dengan akun Google kamu jika diminta.
Buka Google Sheets →
📋 Beri nama bebas, misalnya "Moni Keuangan". Biarkan isinya kosong.
2
Buka Apps Script
Di Google Sheets yang baru dibuka, klik:
Extensions → Apps Script
Tab baru editor kode akan terbuka.
🔧 Hapus semua kode yang ada di editor (biasanya function myFunction(){}), ganti dengan kode di langkah 3.

          4. Execute as: pilih Me

          5. Who has access: pilih Anyone

          6. Klik Deploy

          7. Klik Authorize access → pilih akun Google → Allow
